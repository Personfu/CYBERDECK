# Shared UI Components

This package will contain shared React/UI components for the FLLC CyberDeck interface.

## Planned Components

- `Chip` - Status indicator chips (ONLINE, ACTIVE, DRAFT, etc.)
- `Panel` - Content panel with cyberpunk styling
- `NavBar` - Navigation bar component
- `DataTable` - Styled data table
- `Button` - Action buttons
- `Form` - Form elements with dark theme

## Theme

Cyberpunk ops console aesthetic:
- Uppercase labels
- Green / Cyan / Magenta accent palette
- Black / Graphite panels
- Terminal/console language
