Authorized Use Only

FLLC CyberDeck is an authorized hardware analysis and documentation platform. All activities must be performed with explicit authorization and within defined scope. This platform is designed for legitimate security research, device assessment, and evidence capture.

Prohibited Activities

The following activities are explicitly prohibited:

  • ✗ Exploit execution or vulnerability exploitation
  • ✗ Brute force attacks or credential harvesting
  • ✗ Malware or payload generation
  • ✗ Evasion techniques or stealth operations
  • ✗ Persistence mechanisms or backdoors
  • ✗ One-click attack automation
  • ✗ Unauthorized network scanning or intrusion
  • ✗ Data exfiltration or theft

Permitted Activities

The following activities are supported:

  • ✓ Authorized hardware interface documentation (UART, I2C, SPI, JTAG, eMMC)
  • ✓ Evidence capture and chain-of-custody tracking
  • ✓ Passive IoT device inventory
  • ✓ Session notes and analyst observations
  • ✓ Report generation and PDF export
  • ✓ AI-assisted summarization and drafting (non-offensive only)
  • ✓ Pin mapping and voltage documentation
  • ✓ Firmware version tracking

Authorization Requirements

  • All projects must define an authorization scope
  • Hardware sessions require target identification and authorization verification
  • Evidence artifacts are tracked with SHA256 hashes
  • All actions are logged for audit purposes
  • Reports include authorization statements
